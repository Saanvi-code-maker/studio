# **App Name**: EduSense Learn

## Core Features:

- Personalized Learning Paths: Students can access structured learning modules and content across various subjects, even without an active internet connection.
- Interactive Questioning System: Engage with lessons by answering questions via text input, multiple-choice quizzes, or spoken responses through voice recognition.
- AI-Powered Understanding & Explanations: An AI tool analyzes student responses to identify areas of confusion, partial knowledge, or incorrect guesses, and then generates clear, simple explanations using analogies, short stories, or visual aids to enhance understanding.
- Offline Data Sync with Firebase: Student progress and learning data are stored locally and automatically synced with Firebase when an internet connection becomes available, ensuring continuous learning and data persistence.
- Multilingual Content Support: The app interface and learning content are available in multiple languages to cater to a diverse student body.
- Teacher Insights Dashboard: A simplified view for teachers to monitor student progress, identify common misconceptions, and pinpoint topics where students are collectively struggling.

## Style Guidelines:

- Color scheme: A calm, clarity-focused light theme. Primary elements use a gentle green (#52A65F) to evoke growth and nature, contrasting clearly with a soft background (#EFF9F0).
- Background color: A very light, desaturated green (#EFF9F0) providing a serene and focused learning environment.
- Accent color: A lively chartreuse green (#9ECF46) used for highlighting important information or interactive elements, ensuring visual differentiation and engagement.
- Font: 'Inter' (sans-serif) for both headlines and body text, chosen for its excellent readability and modern, objective feel, suitable for educational content across various device types.
- Use simple, universal icons and supplementary illustrative visuals that complement explanations, making abstract concepts more approachable for students.
- Design layouts to be clean and uncluttered, prioritizing content presentation with ample white space to prevent cognitive overload, and ensuring adaptability across various screen sizes, especially low-end devices.
- Implement subtle, quick animations for feedback on interactions (e.g., correct/incorrect answers) and smooth transitions between screens to enhance user experience without hindering performance on less powerful hardware.
